# Tutorials

Tutorials are **learning-oriented**. They take you through a series of steps to complete a project, building your understanding as you go. The goal is not just to complete the steps, but to learn.

If you're new to `biblealignlib`, start with [Your First Alignment](first-alignment.md).

## Available Tutorials

### [Your First Alignment](first-alignment.md)

A step-by-step guide to installing the library, configuring your environment, loading an alignment set, and exploring verse-level alignments. Suitable for complete beginners.

**You will learn how to:**

- Set up your environment and configure `CLEARROOT`
- Create an `AlignmentSet` and load it with `Manager`
- Navigate verses and inspect source/target tokens
- Display alignment data in a readable format

---

## Jupyter Notebooks

The `docs/tutorials/` directory also contains Jupyter notebooks for interactive exploration:

- `00Index.ipynb` — Overview and navigation
- `01ReadingSourceData.ipynb` — Working with source (manuscript) tokens
- `02ReadingTargetData.ipynb` — Working with target (translation) tokens
- `03ReadingAlignments.ipynb` — Loading and exploring alignment records

These notebooks require a running Jupyter environment with `biblealignlib` installed and `CLEARROOT` configured.
