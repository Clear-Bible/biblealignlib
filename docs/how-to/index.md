# How-to Guides

How-to guides are **task-oriented**. Each guide addresses a specific goal: "how do I do X?" They assume you already understand the basics. If you're new to the library, start with the [tutorial](../tutorials/first-alignment.md) first.

## Working with Alignment Data

- [Load an alignment set](load-alignment-set.md) — configure paths and create a `Manager`
- [Search alignments](search-alignments.md) — find records by source term, target text, or lemma
- [Compare two alignment sets](compare-alignment-sets.md) — diff two versions of the same data
- [Export alignment data](export-alignments.md) — write filtered or merged alignments to JSON

## Automated Alignment

- [Score alignments against a reference](score-alignments.md) — measure precision, recall, and F1
